# CampusNav for SJCE 🚀

A responsive campus navigation web application built for St. Joseph's College of Engineering, Chennai.

This project helps students, parents, visitors, and event participants easily explore the campus, locate destinations, view events, check the mess menu, and navigate using Google Maps.

---

## ✨ Highlights

* Smart campus navigation with Google Maps integration
* Fully dynamic content using JSON (no hardcoded data)
* Event discovery with search and filtering
* Weekly mess menu system
* Responsive UI across all devices
* Dark mode support 🌙
* Feedback system using Formspree

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* JSON (data handling)
* Google Maps (navigation)
* Formspree (form submission)

## 🚀 Features

### 📍 Campus Navigation

* Search places or events
* Filter by categories
* Navigate using Google Maps
* Supports current location routing

---

### 🎉 Event Section

* Fully dynamic using `events.json`
* Search by name or venue
* Filter by type
* Direct navigation to event locations

---

### 🍽️ Mess Menu

* Weekly menu from `mess.json`
* Easy to update without touching UI

---

### 📝 Feedback System

* Built using Formspree
* Ready for real-time submissions

---

### 📱 Responsive Design

* Mobile-friendly layout
* Adaptive navigation
* Clean UI for all screen sizes

## ⚙️ How to Run

⚠️ Do not open directly using `file://`

### Option 1: Python

```bash
python -m http.server
```

Open:

```
http://localhost:8000/clg_nav/
```

### Option 2: VS Code Live Server

Run using Live Server extension
